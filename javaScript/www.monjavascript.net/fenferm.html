L'éditeur JavaScript

Une fenêtre Pop Up qui s'ouvre au chargement de la page et se referme seule après un temps donné

Insérez le code ci-dessous dans votre page Web entre les balises d'entête
<head> ..... </head>
( juste avant </head> )

<script language="JavaScript">
//PLF-http://www.monjavascript.net/
function fenferm(URL,nom,features) {
mafenetre=window.open(URL,nom,features);
setTimeout("mafenetre.close()",5000); // DELAI EN MILLISECONDES
}
</script>

 

Ensuite insérez le code ci-dessous dans la balise <body>

onLoad="fenferm('fenferm2.htm','mafenetre','width=500,height=100')"
EXEMPLE
<body bgcolor="#FFFFFF" background="im/fond.gif" onLoad="fenferm('fenferm2.htm','mafenetre','width=500,height=100')">
 
onLoad="fenferm('fenferm2.htm','mafenetre','width=500,height=100')" chemin/nomfichier (htm, html, gif,...)
