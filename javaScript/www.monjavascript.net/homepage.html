Ci-dessous la 1ère page s'ouvre sur une adresse de votre choix et non sur une page avec des liens

Ensuite Enregistrer la sur votre disque dur (ex: C:\Mes documents\homepage.htm)

Double-cliquer sur le fichier pour l'ouvrir dans votre navigateur, puis enregistrée la comme page de démarrage :

Dans Explorer : menu Outils-Options dans "Page de démarrage" cliquer sur "page en cours"

Dans Netscape : menu Edition-Préférences-Navigator dans "Navigator démarre avec" sélectionner "Page d'accueil" - dans "Page d'accueil" cliquer sur "Utiliser la page courante"
