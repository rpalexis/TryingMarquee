Une Image qui Suit la Souris Script1

Le but est de Cliquer sur JOË.
(Mais un clique franc, il ne faut que le bouton de la souris reste enfoncé)
Attention plus tu cliques sur Joë plus il se méfie.

Insérez le code ci-dessous dans votre page Web entre les balises <BODY> et </BODY> et corrigez le comme suit

<script language="JavaScript">

var y = 0;

function Pos_Souris(p)
{
x = (navigator.appName.substring(0,3) == "Net") ? p.pageX : event.x+document.body.scrollLeft;
y = (navigator.appName.substring(0,3) == "Net") ? p.pageY : event.y+document.body.scrollTop;
}
function clickok()
{
total++;
mess="BRAVO! Tu m'as eu "+total+" fois. Essaye encore si tu peux.";
alert(mess);
distg -= 10;
distd += 10;
if (vit > 5) vit--;
}

{
if (document.getElementById && animate)
{
document.getElementById("idjoe").style.top = posY-40;

Tu peux rendre le jeux plus facile au départ en augmentant la valeur de vit
ou en diminuant la distance entre joé et la souris distg et distd
la distance étant calculé par rapport au coin haut/gauche, distg doit être négatif et sa valeur absolue doit être supérieure, de la largeur de l'image (pixel), de distd
