L'objet Math

L'objet Math est un objet prédéfini, il permet de manipuler les nombres et contient des fonctions mathématiques courantes

    Les fonctions de bases et arrondis de l'objet Math

Méthode description Exemple
abs(Valeur) Retourne la valeur absolue d'un nombre. x = Math.abs(3.2) -> x = 3.2
x = Math.abs(-3.2) -> x = 3.2
ceil(Valeur) Retourne l'entier supérieur le plus proche. x = Math.ceil(3.2) -> x = 4
floor(Valeur) Retourne l'entier inférieur le plus proche. x = Math.floor(3.2) -> x = 3
round(Valeur) Arrondit à l'entier le plus proche. x = Math.round(3.2) -> x = 3
x = Math.round(3.72) -> x = 4
x = Math.round(3.52) -> x = 4
  Pour arrondir à 1 ou plusieurs décimales près x = Math.round(3.52678*10)/10 x = 3.5
x = Math.round(3.52678*100)/100 x =3.53
pow(Valeur1, Valeur2) Retourne le nombre Valeur1 à la puissance Valeur2 x = Math.pow(8,3) -> x = 512
x = Math.pow(16,0.5) -> x =4 (racine carrée)
random() Retourne un nombre aléatoire compris entre 0 et 1 x = Math.random() -> x = 0.8162365933063834 ou 0.5465412569919159 ou ...
sqrt(Valeur) Retourne la racine carrée de la valeur. x = Math.sqrt(16) -> x = 4

    Statistique

Méthode description Exemple
max(Valeur1, Valeur2) Retourne le plus grand des deux entiers. x = Math.max(3.2,3.25) -> x = 3.25
min(Valeur1, Valeur2) Retourne le plus petit des deux entiers. x = Math.min(3.2,3.25) -> x = 3.2

    Logarithmes et exponentielle

Méthode description Exemple
E Retourne le nombre d'Euler Math.E = 2.718 (et quelque chose)
LN2 Retourne le logarithme naturel de 2 Math.LN2 = 0.693 (et quelque chose)
LN10 Retourne le logarithme naturel de 10 Math.LN10 = 2.302 (et quelque chose)
LOG2E Retourne le logarithme de base 2 de E Math.LOG2E = 1.442 (et quelque chose)
LOG10E Retourne le logarithme de base 10 de E Math.LOG10E = 0.434 (et quelque chose)
SQRT1_2 Retourne la racine carrée de 1/2 Math.SQRT1_2 = 0.707 (et quelque chose)
SQRT2 Racine de 2 Math.SQRT2 = 1.414 (et quelque chose)
exp(valeur) Retourne l'exponentielle de la valeur entrée en paramètre x = Math.exp(1) -> x = 2.718 (et quelque chose)
log(valeur) Retourne le logarithme de la valeur entrée en paramètre x = Math.exp(2) -> x = 0.693 (et quelque chose)
    Trigonométrie

Méthode description Exemple
PI Retourne la valeur du nombre PI Math.PI = 3.14159265358979 (et quelque chose)
sin(valeur) Retourne le sinus de la valeur (valeur doit être en radians) sans exemple
asin(valeur) Retourne l'arcsinus de la valeur (valeur doit être en radians) sans exemple
cos(valeur) Retourne le cosinus de la valeur (valeur doit être en radians) sans exemple
acos(valeur) Retourne l'arccosinus de la valeur (valeur doit être en radians) sans exemple
tan(valeur) Retourne la tangente de la valeur (valeur doit être en radians) sans exemple
atan(valeur) Retourne l'arctangente de la valeur (valeur doit être en radians) sans exemple
