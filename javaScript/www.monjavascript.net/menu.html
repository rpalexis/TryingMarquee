Ouverture Dans la Même Fenêtre

Insérez le code ci-dessus dans votre page Web et corrigez le comme suit

<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
function menuderoulant(selection){
window.location=selection.options[selection.selectedIndex].value;
}
//-->
</script>
Cette partie définie la fonction "menuderoulant". Coller le code ci-dessus dans l'en-tête de la page (entre les balises <head> & </head>)


<form name="form1">
<div align="center">
<select name="menu" onChange="menuderoulant(this)">
<option value="#">menu</option>
<option value="menu1.htm">Page 1</option>
<option value="menu2.htm">Page 2</option>
</select>
</div>
</form>

Coller le code ci-dessus à l'endroit où vous souhaitez mettre le menu déroulant.

Indiquer la page à ouvrir : chemin/nomfichier.htm et le nom qui sera indiqué dans le menu


Ouverture Dans Une Autre Fenêtre

<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
function menuderoulant2(url,haut,gauche,largeur,hauteur,options){
window.open(url,"","top="+haut+",left="+gauche+",width="+largeur+",height="+hauteur+","+options);
}
//-->
</script>
Cette partie définie la fonction "menuderoulant2". Coller le code ci-dessus dans l'en-tête de la page (entre les balises <head> & </head>)

<form name="form2">
<div align="center">
<select name="menu2" onChange="choix=this.options[this.selectedIndex].value;if (choix){menuderoulant2 (choix,10,10,480,600,'toolbar=yes,location=yes,status=yes,menubar=yes,scrollbars=yes,resizable=yes')}">
<option value="">menu</option>
<option value="menu1.htm">Page 1</option>
<option value="menu2.htm">Page 2</option>
</select>
</div>
</form>

Coller le code ci-dessus à l'endroit où vous souhaitez mettre le menu déroulant.

Options de la fenêtre : position coin H&G, largeur, hauteur, autres options (pour un pop-up mettre no partout)
Pour une fenêtre environ de la taille de l'écran avec toutes les options:
0,00,screen.width-30,screen.height - 200,'toolbar=yes,location=yes,status=yes,menubar=yes,scrollbars=yes,resizable=yes

Indiquer la page à ouvrir : chemin/nomfichier.htm et le nom qui sera indiqué dans le menu

Ouverture Dans Un Cadre

Insérez le code ci-dessus dans votre page Web et corrigez le comme suit

<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
function menuderoulant(cadre,selection){
eval(cadre+".location='"+selection.options[selection.selectedIndex].value+"'");
}
//-->
</script>
Cette partie définie la fonction "menuderoulant". Coller le code ci-dessus dans l'en-tête de la page (entre les balises <head> & </head>)


<form name="form1">
<div align="center">
<select name="menu" onChange="menuderoulant('parent.frames[\'moncadre\']',this)">
<option value="#">menu</option>
<option value="menu1.htm">Page 1</option>
<option value="menu2.htm">Page 2</option>
</select>
</div>
</form>

Coller le code ci-dessus à l'endroit où vous souhaitez mettre le menu déroulant.

Indiquez le nom du cadre destinataire

Indiquer la page à ouvrir : chemin/nomfichier.htm et le nom qui sera indiqué dans le menu
