Un Menu Déroulant a Deux Listes

Ce menu vous permet de diriger vos visiteurs en fonction d'une catégorie principale et une sous-catégorie

MES AUTRES MENUS        Page Précédente

Insérez le code ci-dessous dans l'entête de votre page (entre les balises<head> ... </head>)


<script language="javascript">
<!--
//PLF-http://www.monjavascript.net/
function process(form)
{
var choix1=form.choix1.options[form.choix1.selectedIndex].value;
var choix2=form.choix2.options[form.choix2.selectedIndex].value;
var adresse=choix1+choix2+".htm";
top.location.href=adresse
}
//-->
</script>
Attention ! la page demandée s'ouvre dans toute la fenêtre. Si vous voulez qu'elle s'ouvre dans la frame où se trouve votre menu, remplacer top.location.href=adresse par location.href=adresse. Si vous voulez qu'elle s'ouvre dans un autre frame remplacer par : parent.maframe.location.href=adresse

 

Insérez le code ci-dessous dans votre page à l'endroit où vous souhaitez mettre les listes de choix et corrigez le comme suit

<form action="" name="Listes">
<div align="center">
<select size=1 name="choix1">
<option value="cine">Cinema</option>
<option value="thea">Théâtre</option>
<option value="spor">Sport</option>
</select>
<select size=1 name="choix2">
<option value=actu>Actualités</option>
<option value=cale>Calendrier</option>
<option value=rese>Réservation</option>
</select>
<input type="button" value="Select" onClick="process(this.form)" name="button">
</div>
</form>

Vous pouvez ajouter des rubriques en insérant des lignes du type : <option value="spor">Sport</option>
Nom de rubrique - début du nom de page - fin du nom de page
Exemple : si vous faites le choix Sport Calendrier vous allez ouvrir une page sporcale.htm
