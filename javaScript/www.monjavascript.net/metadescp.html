<meta name="xxx" ... > Description Nb. Car. Exemple
title Titre de votre site 100 <meta name="Title" lang="fr" content="Titre de votre site">
&
<title>Titre de votre site</title>
identifier-url URL de votre site - <meta name="identifier-url" content="http://www.monsite.com/">
description Description du site en une phrase de préférence. 200 <meta name="description" content="Mon Site - Le site a moi ">
abstract Phrase prise en compte par certains moteur de recherche comme description de votre site. 200 <meta name="abstract" content="Mon Site - L'autre site...">

keywords Liste de mots clés 500
(1000) 		<meta name="keywords" content="javascript, menu, script ">
category Catégorie de votre site (annuaire) - <meta name="category" content="internet">
date-creation-yyyymmdd Date de la création de votre site - <meta name="date-creation-yyyymmdd" content="19990525">">
date-revision-yyyymmdd Date de la dernière modification - <meta name="date-revision-yyyymmdd" content="19990717">
author Auteur du site - <meta name="author" content="Auteur du site">
reply-to Votre e-mail - <meta name="reply-to" content="webmaster@monsite.net">
publisher Celui qui publie votre site - <meta name="publisher" content="monediteur">
copyright Copyright(s) de votre site - <meta name="copyright" content="Infos de Copyright">
generator Logiciels utilisés pour la création de votre site - <meta name="generator" content="Dreamweaver, Flash, Paint Shop Pro">
distribution Définie la destination (public) de votre site Web. - <meta name="distribution" content="global">
robots Diriger le robot
 -

<meta name="robots" content="index, follow">

index : La page sera indexée par le robot.
noindex : La page ne sera pas indexée par le robot.
follow : Les liens contenus dans la page seront indexés.
nofollow : Les liens contenus dans la page ne seront pas indexés.

revisit-after Indique la revisite du robot après n jours - <meta name="revisit-after" content="15 days">
<meta http-equiv="xxx" ... > Description Exemple
refresh 1 - Actualisation automatique de la page <meta http-equiv="refresh" content="60">
refresh 2 - Redirection automatique <meta http-equiv="refresh" content="60; URL=http://www.allhtml.com/">
expires Date limite d'expiration de vos pages (cache) <meta http-equiv="expires" content="wed, 30 sept 1999 12:00:00 gmt">
pragma Interdire la mise en cache <meta http-equiv="pragma" content="no-cache">
set-cookie Inclure un cookie dans le navigateur du visiteur <meta http-equiv="set-cookie" content="cookievalue=xxx; expires=saturday, 25-Sep-99 12:12:30 gmt; path=/">
