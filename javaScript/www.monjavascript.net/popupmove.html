Une Fenêtre Pop Up Qui se Déplace du Bas-Droit de l'Ecran vers le Haut-Gauche

Au Chargement en cliquant sur un Lien ou un Bouton

Ouverture Pop Up          


Insérez le code ci-dessous dans votre page juste avant la balise de fin d'entête </head>

<script language="JavaScript">
//PLF-http://www.monjavascript.net/
function fenetremove(url,largeur,hauteur,options) {
var haut=(screen.height-(hauteur+100));
var Gauche=(screen.width-(largeur+30));
if (haut>Gauche)
haut=Gauche;
else
Gauche=haut;
mafen=window.open(url,"","top="+haut+",left="+Gauche+",width="+largeur+",height="+hauteur+","+options);
for (i=haut; haut>0 ; i--){
if (Gauche==1) break;
Gauche--;
haut--;
mafen.moveTo(Gauche,haut)}
mafen.moveTo(0,0)
}
</script>

Ensuite choisissez parmi les solutions suivantes la façon de l'ouvrir

 

Pour ouvrir le Pop Up lors du chargement de votre page, insérez le code ci-dessous dans la balise <body>

onLoad='fenetremove("pop_pub.htm",480,60,"menubar=no,scrollbars=no,statusbar=no")'
EXEMPLE
<body bgcolor="#FFFFFF" background="im/fond.gif"onLoad='fenetremove("pop_pub.htm",480,60,"menubar=no,scrollbars=no,statusbar=no")'>

Pour ouvrir le Pop Up à partir d'un lien, insérez le code ci-dessous dans votre page

<a href='javascript:fenetremove("pop_pub.htm",480,60,"menubar=no,scrollbars=no,statusbar=no")'>Ouverture Pop Up </a>

Ouverture Pop Up


Pour ouvrir le Pop Up à partir d'un bouton, insérez le code ci-dessous dans votre page

<form>
<input type='button' value='Ouverture Pop Up' onClick='fenetremove("pop_pub.htm",480,60,"menubar=no,scrollbars=no,statusbar=no")'>
</form>

 

Ensuite corrigez comme suit

fenetremove("pop_pub.htm",480,60,"menubar=no,scrollbars=no,statusbar=no")
