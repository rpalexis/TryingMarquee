Afficher une Image dans une Popup
en Neutralisant le Clic Droit

cliquez sur la vignette et l'image s'ouvre dans une popup en interdisant le clic droit.

Cliquez dans cette image et elle se ferme.

Cliquez sur une autre vignette et l'image précédente se ferme avant d'afficher la nouvelle

     

Insérez le code ci-dessous dans l'entête de votre page
(entre les balises<head> ... </head>)


<SCRIPT language="JavaScript">
//PLF- http://www.monjavascript.net/
var fois=0
function AfficheImage(petiteImage,grandeImage,texteImage)
{
document.write('<A HREF="javascript:Affichegrande(\''+grandeImage+'\',\''+texteImage+'\')"><IMG SRC="'+petiteImage+'" HSPACE=0 VSPACE=0 BORDER=0 ALT="cliquez ici pour afficher en grand"></A>');
}

function Affichegrande(cheminImage,texte)
{
newImage = new Image;
newImage.src = cheminImage;
html = '<HTML><HEAD><TITLE>Image</TITLE><meta http-equiv="Pragma" content="no-cache"></HEAD><BODY leftmargin=0 marginwidth=0 topmargin=0 marginheigth=0 oncontextmenu="return false"><CENTER>'+
'<a href="#" onClick="window.close()"><IMG SRC="'+cheminImage+'" BORDER=0 NAME=monImage alt="'+texte+'"border="0" onLoad="window.resizeTo(document.monImage.width+20,document.monImage.height+80); window.moveTo((screen.width-document.monImage.width)/2,5)"> </a></CENTER></BODY></HTML>';
if (fois == 1 ) ouvrirImage.close();
ouvrirImage = window.open('','_blank','toolbar=0,location=0,menuBar=0,scrollbars=0,resizable=0');
ouvrirImage.document.write(html);
fois=1;
}
</SCRIPT>

cliquez ici voir l'image Texte au au passage de la souris sur la vignette

Place l'image à l'endroit voulu : window.moveTo((gauche,haut)

gauche est la distance du bord gauche de l'écran
haut est la distance du haut de l'écran

screen.width est la largeur de l'écran
screen.height est la hauteur de l'écran
document.monImage.width est la largeur de l'image
document.monImage.height est la hauteur de l'image

Pour mettre au centre :
si vous diviser screen.width par 2 vous trouver le centre de l'écran en largeur. Si vous vous soustrayez préalablement la lageur de l'image vous la centrez.
(screen.width-document.monImage.width)/2
Vous faites de même pour la hauteur et votre commande est donc :
window.moveTo((screen.width-document.monImage.width)/2,(screen.height-document.monImage.height)/2)

Pour ne centrer qu'en horizontal:
window.moveTo((screen.width-document.monImage.width)/2,10)
10 étant votre marge du haut

pour en bas à droite :
il ne faut pas diviser par deux (Par contre on ajoute une petite marge)
window.moveTo((screen.width-document.monImage.width)-15,(screen.height-document.monImage.height)-15)

Insérez le code ci-dessus dans votre page Web l'endroit où vous souhaitez installer la vignette et corrigez le comme suit

<script language=JavaScript>AfficheImage("chemin/vignette.gif","chemin/image.gif","Texte sur l Image");</script>
chemin/vignette.gif
chemin/image.gif
Texte sur l Image		 Indiquez le chemin et le fichier de la vignette
Indiquez le chemin et le fichier de l'image en taille normale
Indiquez le texte qui apparaîtra au passage de la souris sur l'image (pas d'apostrophe et autres...)
