OUVERTURE DANS UNE 2EME FENETRE

Pour que la page mensuelle soit ouverte dans une autre page (ou une pop-up) à l'ouverture de votre page:

Insérez le code ci-dessous dans l'entête de votre page (entre les balises<head> ... </head>)

<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
var date_du_jour, heure, minute;
date_du_jour = new Date();
m1 = date_du_jour.getMonth();
m1++;
if (m1 < 10) m1 ="0"+m1;
m1="chemin/lapagedumois"+m1+".htm"
window.open(''+m1+'','','top=100,left=200,width=400,height=200,toolbar=yes,location=yes,'+
'status=yes,menubar=yes,scrollbars=yes,resizable=yes')
//-->
</script>

Indiquer le chemin pour atteindre le page mensuelle
Indiquer le début du nom de la page : Enregistrer chaque page avec ce nom plus le mois en deux chiffre de 01 à 12.
Exemple pour ce mois : " "
Options de la fenêtre : position coin H&G, largeur, hauteur, autres options (pour un pop-up mettre no partout)

Placez votre souris sur le lien et vérifiez la cible dans la barre d'état

Insérez le code ci-dessous dans l'entête de votre page (entre les balises<head> ... </head>)

<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
var date_du_jour, heure, minute;
date_du_jour = new Date();
m1 = date_du_jour.getMonth();
m1++;
if (m1 < 10) m1 ="0"+m1;
m1="chemin/lapagedumois"+m1+".htm"
function ouvfenet() {
window.open(''+m1+'','','top=100,left=200,width=400,height=200,toolbar=yes,location=yes,'+
' status=yes,menubar=yes,scrollbars=yes,resizable=yes')
}
//-->
</script>

Indiquer le chemin pour atteindre votre page
Indiquer le début du nom de la page : Enregistrer chaque page avec ce nom plus le mois en deux chiffre de 01 à 12.
Exemple pour ce mois : " "
Options de la fenêtre : position coin H&G, largeur, hauteur, autres options (pour un pop-up mettre no partout) ne sert pas pour le "lien simple"

Insérez le code ci-dessous à l'endroit où vous souhaitez placer votre lien

Lien simple
<script language="JavaScript">
<!--
//PLF-http://www.monjavascript.net/
document.write('<a href="'+m1+'" target="_blank">la page du mois</a>')
//-->
</script>
_blank : nouvelle page / _parent : même page /...

Ouvrir un lien de votre choix + la fenêtre mensuelle
<a href="votrepage.htm" onClick=ouvfenet()>votre page</a>

Ouvrir dans une autre fenêtre dont vous avez défini les options (dans le script à coller dans le <head>)
<a href="#" onClick=ouvfenet()>votre page</a>
