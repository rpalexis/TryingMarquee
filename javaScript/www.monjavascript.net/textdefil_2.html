Une Fenêtre de Texte Défilant dans Votre Page

Une fenêtre où défilent vos informations sous forme de liens, textes, images, iframes.
Chaque information peut avoir un format différent
(possibilité d'insérer les balises HTML)
Le défilement s'arrête au passage de la souris

Un Texte Défilant Dans Votre Page

Téléchargez textdefil2.js
Transferez le sur votre site
(Sous Firefox le fichier s'ouvre dans une page. Copiez collez dans votre éditeur
et enregistrez-sous textdefil2.js)

 

Insérez le code ci-dessous dans votre page Web l'endroit
où vous souhaitez installer la fenêtre de texte et corrigez le comme suit

<STYLE TYPE="text/css"> Indiquez le format de votre texte
.txt_defil format texte simple
.txt_defil:link format des liens
.txt_defil:visited format des liens déjà visités
.txt_defil:hover format des liens au passage de la souris

var txt_defil_width = 450; largeur de la fenêtre
var txt_defil_height = 100; hauteur de la fenêtre
var txt_defil_bgcolor = '#9966FF'; couleur de fond
var txt_defil_background = ""; image de fond (vide dans l'exemple)
var txt_defil_pause = 5000; Temps entre chaque infos en milliemes de secondes

txt_defil_info[0]='<a href="page1.htm" CLASS=txt_defil>lien1</a> texte 1';
n° de l'information ( indice du tableau txt_defil_info, commence à 0)
liens page1.htm étant la page à ouvrir
Texte de votre lien


txt_defil_info[3]='<iframe src="mon_iframe.htm" width="468" height="60" scrolling="no" frameborder="0"></iframe>';
Ici l'iformation est dans une iframe. Lors de vos test pensez à indiquer une page existante, pour ne pas être redirigé sur une page Erreur 404.
L'Utilisation d'une iframe peut servir à afficher vos pubs
ex: les emplacement rotatifs de NetAffiliation


txt_defil_info[4]='<a href="http://www.monjavascript.net" CLASS=txt_defil> Mon javascript</a><br>'+
'<a href="http://www.monjavascript.net" target="_blank" CLASS=txt_defil> Mon javascript</a><br>'+
'<a href="http://www.monjavascript.net" target="_blank" CLASS=txt_defil> Mon javascript</a><br>';

Si votre ligne est trop longue vous pouvez revenir à la ligne avec le signe + (n'oubliez pas le ' en début et fin de ligne)


Vous pouvez utiliser les balises HTML pour insérer des images, des tableaux, ou mettre en forme le texte de toute ou partie de votre info.

txt_defil_info[1]='<a href="fic.htm" CLASS=txt_defil><div STYLE="color:#FFFF33;" '+
' align="center"><b>Le Jeu de Memory</b></div><br><div align="left">'+
'< table width="300" border="0" CLASS=txt_defil><tr><td><img src="imag.jpg" width="70"'+
' height="51" border="0" align="middle"></td><td>votre texte.</td></tr></table></div></a>';
